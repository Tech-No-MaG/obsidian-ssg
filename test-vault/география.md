# География Мира Аэлис

Мир состоит из трёх основных континентов:

## Континенты:
1. **Северный материк** — здесь расположена [[Столица Империи]]
2. **Южные земли** — территория [[Древнего леса]]
3. **Западный архипелаг** — острова, где живёт [[Кира]]

## Важные места:
- [[Храм Времени]] — древнее сооружение в горах
- [[Порт Штормов]] — крупнейший порт на западе

Связано с: [[История]], [[Расы]]
