# Мана Кристаллы

Природные кристаллы, содержащие магическую энергию.

## Применение:
- Питание для [[Магическая система|магии]]
- Лечение ран
- Создание артефактов

## Месторождения:
- Горы near [[Столица Империи]]
- Глубины [[Древний лес]]
- Подводные пещеры near [[Порт Штормов]]

## Виды:
- Синие (обычные)
- Фиолетовые (редкие, использует [[Элдариан]])
- Чёрные (запрещённые, использует [[Теневой король]])

Связано с: [[Магическая система]], [[География]]
